Aquatest is a self-initiated web-based solution designed to analyze grey water (non-industrial wastewater from household sources) and recommend natural treatment methods based on its pH value. The goal is to promote sustainable water reuse and empower communities with eco-friendly purification practices.

Key Features:
- pH-based analysis: Users input water pH readings to receive tailored treatment suggestions.
- Natural remedies: Recommends plant-based filtration, charcoal beds, sand layers, and other eco-safe methods.
- Educational interface: Explains water safety levels and treatment rationale.
- Responsive design: Built using HTML, CSS, and JavaScript for accessibility across devices.
- Hosted on Vercel: Publicly accessible for feedback and iteration.
